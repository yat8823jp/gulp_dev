#Gulp

my gulp "gulpfile.js"

* watchTask
* webserver

must 

* Node.js
* browser-sync

##install

`npm install -g gulp`

*Macで失敗する場合 sudo npm install -g gulp

###開発ディレクトリにて

ex) htdocs/dev/

初期化

`npm init`

ローカル環境へのインストール

`npm install gulp --save-dev`

###プラグインインストール

画像圧縮

`npm install gulp-imagemin --save-dev`

gulp-webserver

`npm install --save-dev gulp-webserver`

### Browser-Syncインストール

`npm i -D browser-sync`
