ex) htdocs/dev/

###プラグインインストール

必要なプラグインは、package.jsonに記述しています。  
package.jsonに開発に必要なプラグインが記述している場合は、

`npm i`

だけで、ローカルに必要なプラグインが全てインストールされます。
